# React Material Starter

> A simple `ReactJS` starter that uses `Material UI`.

### [preview here](https://mackentoch.github.io/react-material-starter)

## contents
 - react 15.4+
 - [Material UI](http://www.material-ui.com/)
 - Roboto font
 - [Flexbox Grid](http://flexboxgrid.com/)
 - react-hot-loader v3+
